# Search Engine and Information Retrieval (IR) – Course Repository

This repository contains my coursework, notes, assignments, lab programs, and project work related to the **Search Engine and Information Retrieval** course.

The goal of this repository is to organize all learning material and code implementations for easy reference and revision.

---

## 📚 Contents of the Repository

- 📝 Lecture notes
- 🧮 Assignments and solved questions
- 💻 Lab exercises / coding files
- 🔍 Boolean Retrieval model implementations
- 📑 Inverted index creation
- 📊 Ranking and scoring algorithms
- 🧠 Vector space and cosine similarity experiments
- 🧾 IR numericals and practice questions
- 🗂️ PDFs, slides, and reference material
- 🚀 Mini projects related to search engines

## 🛠️ Topics Covered

- Introduction to Information Retrieval
- Search Engines architecture

- Cosine Similarity
- Relevance ranking
- Web search basics
- Crawling and indexing overview
